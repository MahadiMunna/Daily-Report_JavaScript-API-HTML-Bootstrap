Question - 1
What is the difference between regular function and arrow function?

- Regular functions created using function declarations or expressions are constructible and callable. On the other hand, the arrow functions are only callable and not constructible.

Question - 2
How does map, forEach, filter, find works?

- .map() target all the value in an array and can modify them.
- .forEach() method iterates over the array items, in ascending order, without mutating the array.
- .filter() find out all values from an array accordig to condition.
- .find() only find out 1st value from an array according to condition

Question - 3
Why we use template string?

- Template string are created with backtick (`) characters. We use template string for multi-line strings, for string interpolation with embedded expressions, and for special constructs called tagged templates.
